# A summary of TLP datasets introduction

时序链接预测（TLP）所使用的数据主要为**图类型的时间序列数据**，涵盖多个领域，例如通信网络、社交网络、学术引用网络、交易网络等。  

核心的数据字段包括：

- **源节点 ID（source node）**：边的起点节点  
- **目标节点 ID（target node）**：边的终点节点  
- **时间戳（timestamp）**：边发生的时间  

可选字段（视具体数据集而定）：

- **边属性（edge features）**：交互类型、权重等  
- **节点属性（node features）**：用户信息、节点特征等  
- **标签（label）**：用于分类或预测任务
